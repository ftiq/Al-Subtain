## مقدمة للمستخدمين

نظام **إدارة الوثائق والعمليات** هو نظام متكامل لإدارة المراسلات والوثائق الإدارية داخل المؤسسة. يوفر النظام:

- **إدارة المخاطبات**: واردة، صادرة، داخلية
- **الموافقات الهرمية**: نظام موافقات تلقائي حسب التسلسل الإداري
- **المهام الإدارية**: تتبع وتنفيذ المهام
- **سير العمل**: أتمتة العمليات الإدارية
- **الأرشفة**: حفظ واسترجاع الوثائق

### أنواع المستخدمين في النظام:

1. **عارض وثائق**: يمكنه فقط مشاهدة الوثائق
2. **مُساهِم وثائق**: يمكنه إنشاء وتعديل الوثائق
3. **معتمد وثائق**: يمكنه الموافقة على الوثائق
4. **مدير قسم**: يدير وثائق قسمه
5. **مسؤول سجلات**: يدير الأرشيف
6. **مسؤول النظام**: صلاحيات كاملة

## دليل عارض الوثائق

### دورك في النظام:

أنت **عارض وثائق** - يمكنك:
- ✅ مشاهدة الوثائق المخصصة لك
- ✅ قراءة تفاصيل الوثائق
- ✅ تحميل المرفقات
- ✅ متابعة حالة الوثائق
- ❌ لا يمكنك إنشاء أو تعديل الوثائق

---

### مثال عملي 1: مشاهدة المخاطبات الواردة لك

**السيناريو**: وصلتك مخاطبة رسمية من جهة خارجية وتريد الاطلاع عليها.

#### الخطوات:

**1. اذهب للقائمة:**
```
إدارة الوثائق → الوثائق → المخاطبات الواردة
```

**2. أو ابحث عن الوثائق المحالة إليك:**
```
إدارة الوثائق → الوثائق → جميع الوثائق
```
ثم اضغط على فلتر **"المحالة إلي"** من شريط الفلاتر.

**3. ستظهر قائمة بالوثائق:**

| الرقم المرجعي | العنوان | المرسل | التاريخ | الحالة |
|---------------|---------|--------|---------|--------|
| INC/OFL/0001/2025 | طلب استضافة مؤتمر | وزارة التعليم | 2025-01-10 | قيد المراجعة |
| INC/OFL/0002/2025 | دعوة لحضور ورشة عمل | الجهة المنظمة | 2025-01-09 | مسودة |

**4. اضغط على الوثيقة لفتحها:**

ستفتح نافذة تعرض:

```
┌────────────────────────────────────────────────┐
│ 📄 طلب استضافة مؤتمر                          │
│ الرقم المرجعي: INC/OFL/0001/2025              │
├────────────────────────────────────────────────┤
│ المرسل: وزارة التعليم                          │
│ المستلم: أنت (أحمد محمد)                       │
│ التاريخ: 2025-01-10                           │
│ الحالة: 🟡 قيد المراجعة                       │
├────────────────────────────────────────────────┤
│ الموضوع:                                       │
│ طلب استضافة المؤتمر السنوي للتعليم العالي     │
│                                                │
│ المحتوى:                                       │
│ تود الوزارة التنسيق معكم لاستضافة...         │
│                                                │
│ [📎 المرفقات]                                 │
│ - برنامج المؤتمر.pdf                          │
│ - جدول الفعاليات.xlsx                         │
└────────────────────────────────────────────────┘
```

**5. يمكنك:**
- تحميل المرفقات بالضغط عليها
- قراءة جميع التفاصيل
- مشاهدة سجل التغييرات (تبويب "سجل التغييرات")
- إضافة تعليق أو ملاحظة في الدردشة (Chatter) أسفل الصفحة

---

### مثال عملي 2: البحث عن وثيقة معينة

**السيناريو**: تريد البحث عن وثيقة تتعلق بكلمة "مؤتمر".

#### الخطوات:

**1. اذهب إلى:**
```
إدارة الوثائق → الوثائق → جميع الوثائق
```

**2. في شريط البحث في الأعلى:**
اكتب: `مؤتمر`

**3. النظام سيبحث في:**
- العنوان
- المحتوى
- الملخص
- الموضوع

**4. النتائج:**

```
🔍 نتائج البحث عن: "مؤتمر"

✓ INC/OFL/0001/2025 - طلب استضافة مؤتمر
✓ OUT/OFL/0045/2024 - الرد على طلب المؤتمر
✓ INT/MEM/0088/2024 - مذكرة داخلية بخصوص المؤتمر
```

## دليل مُساهِم الوثائق

### دورك في النظام:

أنت **مُساهِم وثائق** - يمكنك:
- ✅ كل ما يمكن للعارض فعله
- ✅ إنشاء وثائق جديدة
- ✅ تعديل الوثائق التي أنشأتها
- ✅ تعديل الوثائق المحالة إليك
- ✅ إرسال وثائق للموافقة
- ✅ إضافة مرفقات وتعليقات
- ❌ لا يمكنك اعتماد أو رفض الوثائق

---

### مثال عملي 2: تقديم الوثيقة للمعالجة

**السيناريو**: أنهيت كتابة المخاطبة وتريد إرسالها للمدير للمراجعة.

#### الخطوات:

**1. افتح الوثيقة التي أنشأتها**

**2. في الأعلى، ستجد أزرار:**

```
┌────────────────────────────────────────────┐
│ [إرسال للموافقة الهرمية] [تطبيق التوجيه]  │
│                                            │
│ 🟢 مسودة                                  │
└────────────────────────────────────────────┘
```

**3. اضغط على:**
```
[إرسال للموافقة الهرمية]
```

**4. ما يحدث تلقائياً:**

```
✓ النظام يبحث عن مديرك المباشر
✓ يُنشئ طلب موافقة تلقائياً
✓ يُبني سلسلة موافقات هرمية:
  1️⃣ مديرك المباشر (خالد أحمد)
  2️⃣ مدير الإدارة (محمد علي)
  3️⃣ المدير العام (عبدالله حسن)
✓ يرسل إشعار للمدير المباشر
✓ تتغير الحالة إلى: 🟡 في انتظار الموافقة
```

**5. رسالة تأكيد:**

```
┌────────────────────────────────────────┐
│ ✅ تم إرسال الوثيقة للموافقة          │
│                                        │
│ طلب الموافقة: #APR-0045              │
│ الموافق الأول: خالد أحمد (مدير مباشر)│
│                                        │
│ [موافق]                                │
└────────────────────────────────────────┘
```

### مثال عملي 3: تعديل وثيقة محالة إليك

**السيناريو**: أحال إليك زميلك وثيقة لإكمال بعض التفاصيل فيها.

#### الخطوات:

**1. اذهب إلى:**
```
إدارة الوثائق → الوثائق → الوثائق المحالة إلي
```

**2. ستجد الوثيقة:**

```
INT/MEM/0099/2025
📝 مذكرة داخلية - تحديث البيانات
👤 المُحال من: سارة خالد
📅 التاريخ: 2025-01-11
```

**3. افتح الوثيقة واضغط على:**
```
[✏️ تحرير]
```

**4. أكمل البيانات المطلوبة**

**5. اضغط [حفظ]**

**6. اترك تعليقاً في الدردشة (Chatter):**

```
[💬 إضافة تعليق]

"تم إكمال البيانات المطلوبة. جاهزة للمراجعة."

[إرسال]
```

**7. النظام سيُرسل إشعاراً تلقائياً للمُحيل (سارة)**

---

### مثال عملي 4: إنشاء مخاطبة داخلية

**السيناريو**: تريد إرسال مذكرة داخلية لزميلك في قسم آخر.

#### الخطوات المختصرة:

**1. إنشاء → المخاطبات الداخلية**

**2. املأ:**
```
العنوان: طلب معلومات عن المشروع الجديد
النوع: مذكرة
المرسل (موظف): أنت
المستلم (موظف): زميلك في القسم الآخر
القسم: قسمك
```

**3. اكتب المحتوى**

**4. حفظ**

**5. لإرسالها فوراً:**
- اذهب لتبويب "سير العمل"
- اضغط [تقديم للمعالجة]

**6. النظام:**
- ✓ يُرسل إشعاراً للمستلم
- ✓ تظهر في قائمة "الوثائق المحالة إليه"

## دليل معتمد الوثائق

### دورك في النظام:

أنت **معتمد وثائق** (موافق) - يمكنك:
- ✅ كل ما يمكن للعارض فعله
- ✅ الموافقة على طلبات الموافقة المخصصة لك
- ✅ رفض طلبات الموافقة
- ✅ تصعيد طلبات الموافقة
- ✅ إضافة تعليقات على الموافقات
- ✅ إنشاء مهام بعد الموافقة
- ✅ اعتماد ورفض الوثائق
- ❌ لا يمكنك إدارة الإعدادات

---

### الفصل 1: الاستلام والتسجيل (موظف الاستقبال - مُساهِم)

## الأسئلة الشائعة

### 1. كيف أعرف إذا كان عندي موافقة معلقة؟

**الجواب:**
- 🔔 ستصلك إشعار
- أو اذهب: الموافقات والأرشيف → الموافقات المعلقة

---

### 2. ماذا أفعل إذا لم أجد الوثيقة؟

**الجواب:**
- استخدم البحث المتقدم
- ابحث بالرقم المرجعي
- تحقق من الفلاتر (قد تكون مخفية)

---

### 3. كيف أعرف حالة وثيقة معينة؟

**الجواب:**
- افتح الوثيقة
- انظر لشريط الحالة في الأعلى
- تبويب "سجل التغييرات" يعرض التاريخ الكامل

---

### 4. هل يمكن تعديل وثيقة بعد اعتمادها؟

**الجواب:**
- لا، الوثائق المعتمدة لا يمكن تعديلها
- يمكن للمدير "إرجاعها لمسودة" (سيلغي الموافقات)
- أو إنشاء وثيقة جديدة مرتبطة بها

---

### 5. كيف أطبع وثيقة؟

**الجواب:**
- افتح الوثيقة
- اضغط زر [طباعة الوثيقة]
- سيُنشأ PDF رسمي بشعار الجامعة
